# Checklist – Instalação do Servidor (Homelab)

Este checklist serve como guia para instalação e configuração do servidor pessoal.

# Hardware
- [ ] PC montado e funcionando
- [ ] BIOS configurada
- [ ] Boot via USB habilitado

# Sistema Operacional
- [ ] Debian instalado (minimal)
- [ ] Usuário criado
- [ ] Atualização inicial do sistema
- [ ] Timezone configurado

# Acesso e Segurança
- [ ] SSH instalado
- [ ] Acesso via chave SSH
- [ ] Login root desativado
- [ ] Firewall (ufw) configurado

# Ferramentas Básicas
- [ ] git
- [ ] curl / wget
- [ ] htop / btop
- [ ] nano / vim

# Docker
- [ ] Docker instalado
- [ ] Docker Compose instalado
- [ ] Usuário no grupo docker

# Serviços Planejados
- [ ] Jellyfin
- [ ] Navidrome
- [ ] Pi-hole
- [ ] Nextcloud

# Backup e Manutenção
- [ ] Estratégia de backup definida
- [ ] Atualizações automáticas
- [ ] Monitoramento básico

# Observações
Espaço para anotações futuras.
