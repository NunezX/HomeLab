# Homelab / Servidor Pessoal

Este repositório documenta meu homelab pessoal, focado em aprendizado prático
com Linux, Docker, Python e desenvolvimento web.

## Objetivos
- Aprender Linux e administração de servidores
- Usar Docker para isolar serviços
- Criar sites e APIs para estudo
- Versionar tudo com Git

## Serviços planejados
- Jellyfin (mídia)
- Navidrome (música)
- Nextcloud (arquivos)
- Pi-hole (DNS / ads)
- Projetos próprios em Python
- Servidor MTA
